En-crooked-ize yourself!

Crooked Teeth is a jQuery plug-in you can use to make page elements crooked. It supports elements containing either background colors or background images.

Essentially, give an element a border and apply Crooked Teeth to it and the object will be made randomly crooked by the border's width amount on each side.

For downloads, source code and more, see the Crooked Teeth github repo.

Usage

Just include the jquery.crookedteeth.js javascript file (along with jQuery) and then call the function with something like $(".myelements").crookedTeeth(). Simple!

Parameters

You can pass the crookedTeeth() method an object containing any of the following attributes:

settingdefaultallowed valuesdescription
amount-An integer or a css-style width shorthand stringThe amount to distort the element by.
xwobbletrueBooleanIf false, no x-axis component to the distortion will be added.
ywobbletrueBooleanIf false, no y-axis component to the distortion will be added.
xsymmetryfalseBooleanIf true, the top and bottom x-distortion will be the same on both sides.
ysymmetryfalseBooleanIf true, the left-side and right-side y-distortion will be the same on both sides
connect-"vertical" or "horizontal"If set it will cause siblings to be connected as specified (see example below)

#2 Padding-based Distortion

This time we'll remove the border and just use padding of 20px all around. Call crookedTeeth() with no arguments and it will use that padding as the distortion amount. Alternately, pass {amount:5} to the function to limit the distortion to just 5 pixels each side:

Using a border is a nice way to avoid having to call crookedTeeth() with arguments. Although this is not always practical, such as when you're using a background image.

#3 Background Images

Here we have 10 pixels of padding, but we could fine-tune the crookedness by passing an amount parameter. When making elements with background-images crooked, you'll find you get a security exception if you load the page as a local html file. It will only work when the page is served by a webserver.

You might want your series of elements to look just slightly more orderly. You can apply symmetry to either axis with by passing a boolean attribute of either {xsymmetry:true} or {ysymmetry:true}. Passing the former will result in this:

You might notice that the first and last elements of the list still have y-axis distortion. You can remove this if needed by passing the {ywobble:0} parameter.
